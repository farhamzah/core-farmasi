## Visual Direction
- Base visual putih bersih.
- Primary accent biru farmasi/fakultas.
- Secondary accent sky/cyan/blue-neutral.
- UI dibuat lebih modern, profesional, dan mudah discan.
- Problem/warning tetap memakai warna status standar dan tidak hanya bergantung pada warna.
- Empty state dibuat lebih jelas.

## Theme/Branding Changes
Filament panel tetap memakai brand `Core Farmasi UBP`, dengan primary color diubah dari amber ke blue. Warna info/success/warning juga dirapikan melalui color config Filament agar status UI tetap konsisten.

## Dashboard Polish
Dashboard default Filament tidak dibuat ulang agar tidak memperbesar risiko. Fokus polish dilakukan pada halaman operasional utama yang sudah ada: Data Quality Dashboard, Import Center, dan App Launcher.

## Data Quality Dashboard Polish
- Menambahkan hero section read-only dengan aksen biru.
- Metric cards diberi hierarchy lebih kuat, shadow halus, dan status badge `OK`/`Review`.
- Section summary menampilkan jumlah metric dan issue group.
- Example lists dibuat lebih rapi dengan blue-tinted panels dan good-state empty state.

## Import Center Polish
- Menambahkan hero section dengan pesan private upload dan no password column.
- Menambahkan lifecycle step cards: template, upload, preview, decision, execute, rollback.
- Template cards diberi blue accent dan required columns panel.
- Upload/preview safety copy diperjelas: preview tidak execute, execute/rollback manual dan membutuhkan konfirmasi.

## App Launcher Polish
- Menambahkan hero section internal navigation dengan badge `Tanpa SSO`.
- App cards diberi blue gradient accent, role badges, disabled state, dan button yang lebih jelas.
- Empty state dibuat lebih informatif.
- Copy tetap menegaskan tidak ada token, SSO, auto-login, atau bypass auth.

## Navigation Polish
Branding dan warna navigasi diperkuat melalui Filament panel color config. Navigation group/label existing dipertahankan agar tidak mengubah kebiasaan admin dan tidak mengganggu test/resource discovery.
